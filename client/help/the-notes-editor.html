Editor Key Words

Certain words typed from within the editor invoke certain actions:

  • .draw - pops up a drawing editor
  • .mc - pops up a medical certificate
  • .dx - To code the consultation with a diagnosis, type .dx followed by whatever you wish to pop up up a list diagnostic terms.
