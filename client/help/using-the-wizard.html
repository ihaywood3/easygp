Using the Wizard

Our setup wizard will hopefully make it easy for you to create your general practice organisation and add your initial system admin user. If not, please let us know how to improve it!

There are a number of steps to complete

  • The Introduction
  • Accept the licence agreement
  • Creating your practice organisation
  • Creating a system administrator
  • Reviewing your configuration
  • Save your configuration
  • Running EasyGP for the first time
