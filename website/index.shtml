About EasyGP

EasyGP is an open-source medical records program for Australia. Despite the name it is suitable for specialist practice also.

Status

EasyGP is stable, feature-complete, and has been in daily use at 3 separate sites (2 GP, one specialist) for 4 years.

However, the main aim of this site is to recruit more developers. We urge anyone to discuss with us before proceeding to clinical use. We are not able to provide commerical support at this stage, you would need to be (or have access to) an experienced UNIX/Linux administrator to install and maintain the software.

Features

  • Prescribing using the PBS Yellow Book plus common private-only items.
  • Notes including diagrams
  • Obstetric module
  • Highly automated careplans and mental-health plans
  • Billing, private and bulk-billing
  • ...and pretty much any feature in proprietary products

What's Missing

The "Onlines": Medicare Online billing, online prescribing, and PCEHR.

Platform

Server runs on PostgreSQL, client is a Linux-native graphical client (although it has been successfully used on Windows clients using VirtualBox.)
